🌟 প্রজেক্ট ওভারভিউ

Comilla Seba হলো কুমিল্লা জেলার জন্য তৈরি একটি আধুনিক **লোকাল সার্ভিস প্রোভাইডার ডিরেক্টরি প্ল্যাটফর্ম**। এখানে ইলেকট্রিশিয়ান, প্লাম্বার, কার্পেন্টার, মেকানিক, পেইন্টার, AC টেকনিশিয়ানসহ বিভিন্ন পেশাজীবীদের খুঁজে পাওয়া যায় সহজেই।
